ℹ️ About This Project

The Gamified Study Tracker is a web application designed to make studying fun and motivating for students. It allows students to add study tasks, earn points by completing them, and claim rewards. The app also helps track daily goals and student profile information.

  • 📝 Add tasks with custom points
  • ✅ Mark tasks as complete
  • 🎁 Claim rewards using earned points
  • 👤 Store student profile details

This project integrates HTML, CSS, JavaScript on the frontend, with Node.js, Express, and MySQL on the backend.
